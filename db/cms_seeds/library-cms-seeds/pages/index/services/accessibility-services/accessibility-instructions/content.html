Accessibility Instructions for Specific Library Resources

Steps for Students to Access the Internet Archive Print Disabilities Collection

  1. If you don’t already have one, please create an Internet Archive account at https://archive.org/account/signup
  2. Notify digitization@ualberta.ca or your Accessibility Advisor directly. Provide your Internet Archive username and the email address you used to register.
  3. Once you’ve received confirmation that your account has been confirmed, visit IA’s Print Disabled Collection visit: https://archive.org/details/printdisabled?sort=-reviewdate
  4. When you find an item you would like to borrow (for example: https://archive.org/details/lordofringsonevo00jrrt ), click “Read Book”
  5. To read the book you have several options:
    1. Read the book onscreen: there are tools to enlarge the text, or read the book aloud through your browser.
    2. Download an encrypted Adobe EPub file and read with your computer, laptop, or mobile device.
      (requires you to install Adobe Digital Editions: https://www.adobe.com/ca/solutions/ebook/digital-editions/download.html)
    3. Download an encrypted Adobe PDF and read with your computer, laptop, or mobile device.
      (requires you to install Adobe Digital Editions: https://www.adobe.com/ca/solutions/ebook/digital-editions/download.html)

You can also request a different file format if these don’t work for you. Please email digitization@ualberta.ca to request a file.


Steps for Students to Request HathiTrust Materials

  1. Make sure you are registered with Accessibility Resources.
  2. Search the HathiTrust digital library, and ensure you have unchecked the “full view” checkbox, and are searching “All items” and not just “full view” items
    1. Many items will already have a full PDF available to view. If you require a download of the entire item you can complete the Google form to request a copy of the file. Click on the catalogue record and then copy the URL for the item you’d like to request.
    2. For restricted items that only have the “limited (search-only)” note, click on the catalogue record and then copy the URL for the item you’d like to request.
  3. Submit the request to the Libraries via this Google form
    1. The form collects user name, phone number, email and student ID, as well as Hathi Trust URL, and optional publication information
  4. The Library will access a copy of the full PDF and will email it to you within two working days
