Creating OER

Create your own course e-textbook using Pressbooks!

Some features of our Pressbooks platform include:

  • Using interactive activities, like videos from YouTube, quizzes created with H5P, and social annotation using Hypothesis.
  • Integration with Canvas for sharing content with students.
  • Co-authorship options with an easy to use authoring tool.
  • Simple updating and editing tools, to help with future course needs.

Eligibility for this service requires:

  • The creation or adaptation of content that will result in openly licensed learning materials.
  • The creation or adaptation of content that will be used in a course.
  • The primary creator or adaptor must be a faculty/staff at the University of Alberta; or affiliated with a partner institution.
  • The work requires adhering to accessibility guidelines.
