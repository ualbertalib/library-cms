What is a Library Services Platform (LSP)?

Library Services Platform (LSP) is the name for the next generation of ILS (Integrated Library Services) applications. LSPs are built using more modern technology and frameworks. LSPs follow a Software as a Service (SaaS) model, which means that LSPs are cloud hosted and run on a subscription basis.

Services which will be impacted by the move from an ILS to an LSP include:

  • All functions of the ILS including Workflows
  • Circulation
  • Cataloging
  • Discovery
  • Electronic Resource Management
  • Any service which connects to our catalogue, or SirsiDynix

Services which may be assessed for inclusion include:

  • Interlibrary Loans (ILL)
  • Acquisitions
  • Website Content Management System (CMS)

More Information

For a more detailed explanation of LSPs please check out Smart Libraries Q&A: Differences between ILS and LSP by Marshall Breeding or reach out to Ivan Chan (LSP Project Manager).

The LSP Selection Working Group is still being formed and we are only in the initial stages of the LSP migration project. We will share with you more information about the project in coming updates.
