Z39.50

Production Server

Server: ualapp.library.ualberta.ca (129.128.216.67)

  • Port: 2200
  • Database name: Unicorn

Errors

Some searches produce errors identified by their internal Sirsi codes:

  • when searching for a common keyword (such as "Canada"), you may receive this error: "[2] Temporary system error -- v2 addinfo '#DO$(59) (Unicorn error: 59)'". This appears to happen when the result set is too large. What constitutes "too large" appears to vary depending on the load on the server.
  • if you attempt to browse the general keyword index (Use attribute 1016), or to search it for an unanchored phrase (Position attribute 3), the server will immediately drop the connection.

Available searches:

The NEOS Z39.50 server supports almost all of the searches specified in the Bath Profile, version 1.1, functional area A, levels 0 and 1. Some other bib-1 searches also work, in addition to the official Bath Profile searches.
