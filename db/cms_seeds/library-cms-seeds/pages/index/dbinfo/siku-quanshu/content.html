Users will need to download a client program into their own personal computer or laptop while on campus using UWS. An Installation Guide and the database User Manual are also available and they include important information regarding the technical requirements for user's PCs, especially required language packages.

After you have downloaded the client program, please select “IP Address” to login.

NOTE:

  • Installation of the Client Program is necessary for access.
  • Access is available from on-campus only.
  • The database is not compatible with Apple operating systems, but it will work on Apple computers running Windows.
