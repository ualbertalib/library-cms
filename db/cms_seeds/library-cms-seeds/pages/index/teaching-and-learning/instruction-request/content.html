Library Instruction Request Form

Please complete this form to request library instruction. To request multiple sessions, please fill out the form separately for each session. Sessions usually need to be scheduled two to three weeks in advance. Earlier requests are always appreciated and easier to schedule.

We ask that course instructors be present during library instruction.

Note: Bruce Peel Special Collections does not offer customized instruction. Workshop information is available on their website.

If you have questions about this form, please contact library.teachingandlearning@ualberta.ca.

Formulaire de demande de formation en Bibliothèque

Veuillez remplir ce formulaire pour demander une formation en bibliothèque. Pour faire une demande pour plusieurs sessions, veuillez remplir le formulaire séparément pour chaque session. Les sessions doivent généralement être programmées deux à trois semaines à l'avance. Les demandes anticipées sont toujours appréciées et plus faciles à programmer.

Les professeurs doivent être présents dans la salle de classe pendant les séances de formation.

Remarque : La bibliothèque de Collections Spéciales Bruce Peel ne propose pas de formation personnalisée. Les informations sur les ateliers sont disponibles sur leur site web.

Si vous avez des questions concernant ce formulaire, veuillez contacter library.teachingandlearning@ualberta.ca.
