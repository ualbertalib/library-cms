Programme d'indicateur du coût zéro des manuels scolaires et REL

{{ cms:snippet oer-menu-french }}

Pour les instructeurs : Enregistrez votre cours dans le programme ZTC (en anglais)

Pour les étudiants : Comment identifier les cours ZTC dans les Bear Tracks (en anglais)

Pour soutenir le travail de plaidoyer de l'Union des étudiants visant à mettre en évidence diverses options de manuels scolaires à faible coût pour les étudiants, l'Université a mis en œuvre le matériel de cours ZTC (Zero Textbook Cost) au cours de l'année universitaire 2021/22 en tant que projet pilote. Le programme encourage les instructeurs à enregistrer leurs cours ZTC afin qu'ils puissent être signalés dans Bear Tracks. Les étudiants peuvent alors utiliser le ZTC comme paramètre de recherche lorsqu'ils prennent des décisions concernant leurs cours.

"Coût zéro du manuel scolaire" signifie qu'un étudiant n'encourt aucun coût pour accéder au matériel de cours et constitue un excellent moyen d'accroître l'accessibilité du matériel d'apprentissage pour tous les étudiants. Les options peuvent inclure un mélange de ressources éducatives libres (REL) ou de documents de bibliothèque sous licence qui peuvent être liés par une liste de lecture. Si vous êtes un enseignant intéressé par l'exploration d'options alternatives de matériel de cours, la bibliothèque offre un service de recherche lié à la page Trouver des REL. Si vous êtes un étudiant, demandez à votre professeur de vous parler de ZTC ou contactez l'Union des étudiants.

Pour plus d'informations sur les avantages de ce programme, un rapport sur les économies estimées des cours ZTC pendant l'année universitaire 2021/2022 a été préparé par l'Union des étudiants.
