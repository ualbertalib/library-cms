Tip: * To see members of a specific group, create a 'fake' event using the calendar (you don't have to save it). Enter the email address for the group the same way you'd add any guest. Next to the Group in the guest list, click the Arrow to see group members. If you don't see the group icon and arrow, you don't have permission to view the group's member list.
