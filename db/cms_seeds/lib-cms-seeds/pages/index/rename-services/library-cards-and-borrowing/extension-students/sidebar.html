Access to Online Resources

Remote access to library licensed online resources is available to Extension students enrolled in some programs. Please enquire at the Faculty of Extension for details.

All Extension students may access library licensed online resources on-site. To access online resources via library computer terminals or via the U of Alberta wireless network, obtain a Temporary Network ID by presenting your photo ID with proof of a Canadian address at any library Service Desk. For more info see: Temporary Network IDs.
