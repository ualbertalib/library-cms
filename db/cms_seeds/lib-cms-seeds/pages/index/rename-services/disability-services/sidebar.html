Computer Facilities, Hardware, and Software

Find available software in IST computer labs across campus (e.g., ZoomText, Jaws)

The Kurzweil Reading Edge machine scans and reads printed material and is available in the microreading room on the 2 nd floor of Rutherford North. The latest Kurzweil software is available in IST computer labs.

Some computer stations in the libraries have some accessibility software (i.e, magnifier, narrator, on-screen keyboard). Contact specific libraries, if you have questions.
