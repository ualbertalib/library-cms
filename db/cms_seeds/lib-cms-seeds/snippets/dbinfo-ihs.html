Access to {{ cms:text title }} through IHS Engineering Workbench is restricted to University of Alberta registered faculty, staff and students. Access requires login with a current CCID (Campus Computing ID) as well as individual registration on the product site.
Before using these standards, please read the Conditions of Use below:

  1. Enter your CCID and Password to access the Account Login page.
  2. If you are already registered with IHS, enter your email address and password. Otherwise, you must register with IHS before proceeding.
  3. Select ‘IHS Engineering Workbench’
  4. Ensure that the ‘My Subscription’ box is checked.
  5. To restrict your search to only {{ cms:page:label:string }} standards, type "{{ cms:page:label:string }}" (without the quotation marks) in the Document Number box, and click the Search button.
  6. The resulting set will include all {{ cms:page:label:string }} standards to which we have full-text access. Any further keyword or document number search will be limited to the {{ cms:page:label:string }} standards only.
